# Game Play Analysis System
게임 플레이 분석 및 추천 시스템

## 프로젝트 개요
username 입력 시 사용자 계정 정보에서 5개의 feature를 추출하여 플레이 스타일 분석 및 자신의 플레이와 비슷한 프로게이머 선수 결과 제공 (LoL 한정)

## 개발 기간
2023.08.14 - 2023.08.25

## 개발 환경 및 언어
- `VanilaJS`
- `Python`
- `Mysql`
- `VScode`
- `Figma`
